# 🍅 Tomato - Food Delivery Web App

A full-stack **Food Delivery Web Application** built with the **MERN Stack**, designed to provide a smooth and modern food ordering experience for users and an efficient management dashboard for admins.

This project includes a **customer-facing frontend**, a powerful **admin panel**, and a scalable **backend API** for managing foods, menus, carts, and orders.

---

##
👉 Frontend: https://food-delivery-web-frontend-xi.vercel.app/
👉 Admin UI: https://food-delivery-web-admin.vercel.app/orders

## 🚀 Live Features

### 👨‍🍳 User Side
- Browse food items dynamically from the database
- Explore menu categories
- Search for foods
- Add items to cart
- Update cart quantities
- Place food orders
- Track order status
- Clerk-based user authentication

### 🛠️ Admin Panel
- Add new food items
- Add menu categories
- Upload images using Cloudinary
- View all food items
- View all menu categories
- Remove food/menu items
- View all customer orders
- Update order status
- Cancel/remove orders

### ⚙️ Backend Features
- REST API architecture
- MongoDB Atlas integration
- Cloudinary image upload
- Clerk user authentication support
- Cart and order management
- Separate models for:
  - Users
  - Food
  - Menu
  - Cart
  - Orders

---

## 🧰 Tech Stack

### Frontend
- React.js
- React Router DOM
- Context API
- Axios
- Tailwind CSS
- React Toastify
- Clerk Authentication

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- Multer
- Cloudinary
- JWT / Admin Auth Middleware

### Database & Services
- MongoDB Atlas
- Cloudinary
- Clerk
